# 3 . 엄격모드 strict mode

Strict Mode 를 쓰면 2번씩 컴포넌트가 실행됨으로써 오류를 찿기 쉽다.
```jsx
ReactDOM.createRoot(document.getElementById('root')).render(

    <StrictMode>

        <App />

    </StrictMode>

);
```
