# 토큰 url 

url 이 IFPS에 저장되어있다 그래서 올라가있는것 

> 그러면 어떻게 생성할까 IFPS 링크 주소는?? 

피나타에서 생성 가능

똑같은 이미지를 올릴려고 하려면 막힌다 

NFT 발행방법 2가지 

1. 외부 데이터에 이미지 값을 저자하고 그 주소 정보만을 저장하는 방식 
2. 랜덤값을 생성해서 NFT 값에다 넣어준다 (토큰 아이디가 랜덤이 되는 것 )

게임에서는 골드나 아이템등을 교환한다 

그럴때 쓰는 것은 ERC1155 

이것은 ERC20과 ERC721을 동시에 관리
