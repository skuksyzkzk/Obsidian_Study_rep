# 재귀함수의 기본적인 이해

재귀함수는 실행 중간에 다시 재퀴함수가 호출되면, 함수의 복사본을 더 만들어서 복사본을 실행한다

### 재귀함수는 탈출조건이 중요하다

```c
#include <stdio.h>

int Fac(int n) {
	if (n == 0)
		return 1;
	else
		return n * Fac(n - 1);
}

int main(void) {
	printf("4! = %d\n", Fac(4));
}
```
